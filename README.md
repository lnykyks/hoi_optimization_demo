# HOI Optimization Demo

A repository contains HOI-optimization-demo. The hoi(hand object interaction) optimization pipeline is a part of Robowheel[https://zhangyuhong01.github.io/Robowheel/], which improves contact plausibility, reducing penetration, and refining hand-object alignment.
